# Script to Video

AI-powered script to video generation system.

## Features

- 📄 Script input and analysis
- 🎨 Character and scene design
- 🎬 Storyboard editing
- 🎥 Video generation
- 🌐 Web deployment ready

## Quick Start

### Local Development

```bash
# Start backend
cd backend
pip install -r requirements.txt
python main.py

# Start frontend (new terminal)
cd frontend
npm install
npm start
```

### Web Deployment

See [DEPLOY.md](./DEPLOY.md) for detailed deployment instructions.

## Tech Stack

- Frontend: React + Material-UI
- Backend: FastAPI + Python
- Deployment: Vercel + Render
