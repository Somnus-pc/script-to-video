# 剧本转视频系统 - 设计文档

## 项目概述
一个本地运行的客户端应用，将剧本自动转换为分镜视频。

## 核心功能模块

### 1. 剧本输入模块
- 支持文本直接输入
- 支持文档上传（.txt, .docx, .pdf）
- 大文本处理能力

### 2. 剧本分析模块
- 使用大模型分析剧本结构
- 提取人物角色
- 提取场景信息
- 生成人物/场景描述

### 3. 三视图生成模块
- 根据描述生成人物三视图（正面、侧面、背面）
- 生成场景概念图
- 支持多种生图模型

### 4. 分镜拆分模块
- 将剧本拆分为分段
- 为每段生成分镜提示词
- 支持对话式修改分镜提示词

### 5. 分镜图生成模块
- 根据提示词生成分镜图
- 支持多种生图模型

### 6. 分镜视频生成模块
- 图生视频
- 支持多种视频生成模型

### 7. 预览编辑器
- 合并预览所有分镜视频
- 调整分镜内容和顺序
- 一键合并导出长视频

## 技术架构

### 前端
- React + TypeScript
- Tailwind CSS
- 视频播放器组件

### 后端
- Python FastAPI
- 文件处理
- API 代理

### AI 模型接入

#### 文本分析模型（免费/可选）
1. **Kimi** (moonshot) - 长文本能力强
2. **GLM** (zhipu) - 中文理解好
3. **本地模型** - Ollama/LM Studio

#### 图像生成模型
**免费：**
1. **Pollinations.ai** - 完全免费
2. **Leonardo.ai** - 免费额度
3. **Stable Diffusion XL** - 开源

**付费：**
1. **Midjourney** - 高质量
2. **DALL-E 3** - OpenAI
3. **Stable Diffusion API** - 各种服务商
4. **通义万相** - 阿里云
5. **即梦** - 字节跳动

#### 视频生成模型
**免费：**

2. **即梦** - 免费额度
3. **Runway Gen-2** - 免费额度
4. **Pika** - 免费额度

**付费：**
1. **可灵 Pro** - 高质量
2. **Runway Gen-3** - 专业级

## API Key 配置
所有付费模型都提供 API Key 配置入口，存储在本地配置文件中。

## 数据流
```
剧本输入 → 文本分析 → 提取人物/场景 → 生成三视图
                                      ↓
                         分镜拆分 → 生成分镜提示词
                                      ↓
                         对话修改 → 确认分镜提示词
                                      ↓
                         生成分镜图 → 图生视频
                                      ↓
                         预览编辑器 → 调整/导出
```
