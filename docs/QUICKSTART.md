# 快速开始

## 5 分钟上手

### 1. 一键启动
```bash
# Windows
start.bat

# Mac/Linux
./start.sh
```

### 2. 配置 API Key（首次使用）
1. 打开 http://localhost:3000
2. 点击右上角 "API 配置"
3. 填入 Moonshot (Kimi) API Key
   - 获取地址：https://platform.moonshot.cn/
   - 新用户有免费额度
4. 点击保存

### 3. 输入测试剧本
复制以下测试剧本到输入框：

```
《雨夜相遇》

场景：城市街道 - 夜晚 - 雨夜

林小雨（25岁，女，白领）独自撑着伞走在雨夜街头，神情落寞。
突然，一把黑伞遮住了她的头顶。

陈默（28岁，男，设计师）站在她身旁，目光温柔。

陈默："又忘记带伞了？"

林小雨抬头，眼中闪过惊讶和欣喜。

林小雨："你怎么知道我在这里？"

陈默微笑，将伞向她倾斜。

陈默："因为我知道，下雨天你总会走这条路。"

两人相视而笑，在雨夜中并肩前行。
```

### 4. 开始创作
1. 点击"开始分析"
2. 查看分析结果（人物、场景、分段）
3. 点击"下一步"进入人物设计
4. 为林小雨和陈默生成三视图
5. 进入分镜编辑，生成分镜图
6. 最后生成视频并导出

## 功能说明

### 剧本输入
- 直接粘贴文本
- 或上传 .txt / .docx 文件

### 剧本分析
自动提取：
- 人物角色（姓名、性格、外貌）
- 场景信息（地点、时间、氛围）
- 剧本分段

### 人物/场景设计
- 为每个人物生成三视图（正面、侧面、背面）
- 为每个场景生成概念图
- 支持多种生图模型

### 分镜编辑
- 自动为每段剧本生成分镜提示词
- 支持手动编辑提示词
- 支持 AI 对话修改（如："让镜头更远一些"）
- 一键生成分镜图

### 视频生成
- 选择视频生成模型
- 为每个分镜生成短视频
- 预览所有分镜
- 一键合并导出完整视频

## 使用技巧

### 1. 提示词优化
- 分镜提示词可以用自然语言描述修改
- 例如："改成夜景"、"增加动作感"、"让画面更温馨"

### 2. 模型选择
- **图像生成**：
  - Pollinations.ai - 免费，速度快
  - DALL-E 3 - 质量高，理解力强
  - 通义万相 - 中文友好

- **视频生成**：
  - 可灵 - 动作流畅
  - Runway - 质量专业
  - Luma - 效果逼真

### 3. 成本控制
- 先用免费模型测试效果
- 满意后再用付费模型生成最终版本
- 视频生成是最耗费的环节，可以只生成关键分镜

## 示例工作流程

```
1. 输入剧本 → 2分钟
2. 分析剧本 → 30秒
3. 设计主要人物 → 5分钟
4. 生成分镜图 → 10分钟
5. 生成视频 → 20分钟
6. 合并导出 → 2分钟

总计约 40 分钟完成一个短片
```
